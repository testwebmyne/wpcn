介绍：
WordPress 表情秀这个插件主要用于把留言中的一些关键字转换为对应的图片，目前采用的图片是悠嘻猴系列的图片，你可以根据你个人爱好改成兔斯基，洋葱头等系列头像图片都可以。

1.2 版本增加一个开关也能替换文章中的关键字，把插件文件中的 $convert_content_keywords 的值设为 1 即可替换文章中的关键字。也开始支持 Win 主机。

特征：
1. 让你的博客更加个性化和生动化。
2. 屏蔽一些或者你不想看到或者粗俗留言。


安装

1. 解压缩
2. 把文件夹 emotions 上传到你的 WordPress 博客的 wp-content 目录下。
3. 把文件 wp_emotion_show.php 上传到你插件目录下，激活即可。
4. 查看你的留言中的关键字是否转化成图片，如果没有，尝试输入一些关键字。

描述
这个插件可以无限添加关键字（注意关键字太多会影响你博客的速度！），这个插件是通过读取 emotions 文件夹中的图片的名字作为关键字的。所以你只需上传图片到 emotions 文件夹下，并把图片的名字命名为你想要的关键字即可。

当你停用这个插件之后，你的留言中的关键字又归为正常。
