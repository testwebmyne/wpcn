== Description ==

<p>WordPress 2.3 statistics Plugin will add the statistics into your blog, such as total posts, pages, tags.</p>
<p>This plugin support WordPress Widgets and Sidebar Modules. You can put the statistics into your blog page easily.</p>


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Manage > Option > wp-statistics to configure plugin output.
4. Navigate to themes > widgets to add the statistics into your blog.
